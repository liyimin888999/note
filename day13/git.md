### git

分布式的版本管理系统

[git官网](https://git-scm.com) 

#### git基础操作

1. git status 

   查看状态

2. git add -A
   * 后面加 -A 是全部提交
   * 后面加某个 文件名 是只提交某个文件

3. git commit -m "提交时的说明,比如第一次提交项目"

   commit到本地仓库

4. git push origin master 

   提交代码到远程仓库

git log 查看日志；退出时按键盘 Q 。

git chcekout 文件名：放弃文件修改
